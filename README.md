
# Ambiente de trabalho (Ubuntu 14.04)

Segue algumas configurações para se realizar no ambiente de trabalho.   

## Terminal

### Alias

Edite o arquivo `~/.bashrc` 

    alias ..='cd ..'
    alias ...='cd ../../'
    alias ls='ls $LS_OPTIONS'
    alias ll='ls $LS_OPTIONS -alFh'
    alias l='ll'
    alias la='ls $LS_OPTIONS -A'
